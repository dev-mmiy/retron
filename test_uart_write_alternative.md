# UART書き込み代替案

## 問題

キャッシュフラッシュを追加しても、UARTDRへの書き込みが反映されません。

## 代替案

### 方法1: QEMUのmonitorで直接書き込みをテスト

QEMUのmonitorを使用して、UARTDRに直接書き込んでテストします：

```
(qemu) x/16x 0x09000000
(qemu) memwrite 0x09000000 0x48 4
(qemu) x/16x 0x09000000
```

### 方法2: 別のUARTアドレスを試す

QEMU virt machineでは、PL011 UARTが別のアドレスにマッピングされている可能性があります。

### 方法3: QEMUのシリアル出力設定を変更

`-chardev`オプションを使用して、シリアル出力を別の方法で設定します。
